Proyecto Academico Universitario

Proyecto Segunda Pre-Entrega

Buenas! Esta pagina esta en proceso de construccion dado a que esta es la segunda pre-entrega del proyecto final. Sin embargo, a continuacion me gustaria comentar de lo que se basará el proyecto

La idea de este proyecto es crear una página universitaria para ayudar a los profesores a colocar las notas de manera virtual, de este modo los alumnos podrian recibir una notificación cuando tengan disponible sus respectivas notas y una vez que todas las notas esten disponibles o haya terminado la cursada, el programa permitá calcular si los alumnos pudieron promocionar (siempre y cuando sea posible promocionar), si aprobaron pero deben rendir un examen final, si deben rendir recuperatorio o si deben recursar la materia.

Ademas, la idea es que haya varias materias, cada materia debe tener espacio para poner la nota del primer examen, la nota del segundo examen, la nota de recuperatorio, del examen final, el promedio y el informe de la cursada (Promocion, recuperatorio o recursar).

Por último, me gustaria aclarar que esta pagina fue creada solamente para la segunda pre-entrega. Esta NO será la página del proyecto final.
